# chatgpt-web
ChatGPT商业版，支持AI绘画、卡密兑换、KEY轮询、用户试用（可任意配置）、用户推广等强大功能


## 联系

Q群2：159740513

## 功能

- [x] 登录、注册、用户管理

- [x] 用户试用（可任意配置，给客户试用机会，用过才说好）

- [x] 用户自推广（让用户去推广咱们的ChatGPT，省去拓客烦恼）

- [x] 服务端保存用户聊天记录

- [x] AI绘画（更加强大的Mj）

- [x] key 轮询池

- [x] 用户额度管理

- [x] 自定义回复内容

- [x] 自定义敏感词

- [x] 详细安装教程，完善售后

- [ ] 更多..
